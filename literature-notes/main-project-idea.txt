This project will explore the notion that declarative memory (in humans) can serve as a basis for aesthetic judgement. We invoke a theory of decision making called instance-based learning as a macro theory of how one uses past instances, encoded in memory, as generating key information for a future decision.  And, we invoke Anderson's theory of needs probability of memory which has the supposition that the accessability of a specific memory instance is directly related to prior systematic statistical patterns, typically temporal, in the environment. (Anderson may be assuming an ergodic environment).  And, we are wondering how the inverted-u theory of stimulus complexity, coming from the theory of experimental aesthetics, might serve us in thinking about how memory might serve as the basis of aesthetic preference.  Is there a way to integrate all of these ideas?  I wonder.  You will see the References listed below for all of these topics. 

This project is, fundamentally, attempting to build a computational model of aesthetic judgement.  The computational basis is pyactup (see the file ./pyactup-info.text for details), a model of declarative memory encoding and retrieval.  In our initial theoretical sketch, each encoding of an instance (think instance-based learning) is implemented in pyactup (think ACT-R declarative memory).  Further, we will invoke the notion of a chunk in memory as the representation for each instance.  A chunk is a set of slot-value pairings where a slot is a variable and the value is the value of that variable for a given chunk.  Each IBL instance is encoded as a chunk.  And, by Anderson's memory theory, when an agent/person is confronted with a piece of music or a song, the agent can compute the difference between its activation in memory and the chunk in memory with the highest current activation.  This difference is the kernel of aesthetic judgment/evaluation.
